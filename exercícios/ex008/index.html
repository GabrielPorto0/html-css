Principais formatações:

Negrito/destaque:

Nessa frase temos um termo em negrito usando a tag <b>

Nesta frase, tenho um termo em destaque

Italico ou enfase:

Nessa frase temos um termo em itálico usando a tag <i> (não semantica)

Nesta frase temos um termo em enfâse usando a tag <em>(semântica)

Mark ou Marca-texto:

Nesta frase temos um termo marcadousando a tag <mark>

E no outro parágrafo, temos outro termo marcado

Texto grande e pequeno:

Estamos criando um texto grande e um texto pequeno nesse parágrafo, usando a tag <small>

Texto deletado:

Podemos marcar um texto como excluído para indicar que ele pode ser lido, mas não considerado usando a tag <del>

texto inserido:

Podemos marcar um texto inserido para dar uma enfâse e indicar que ele foi adicionado depois usando a tag <ins>

Existe também o texto sublinhado mas de maneira não semantica usando a tag <u>

Texto sobrescrito:

Para inserir coisas do tipo x20+3 usamos a tag <sup>

Texto Subscrito:

Para inserir coisas do tipo h20 usamos a tag <sub>
